
## How to use this Code

Clone Facedetect repository to your computer<br />
go to "cpp/bin"<br />
run ./detectFaceFlip ../RawImage\ 1.jpg 0   for face detection<br />
run ./detectFaceFlip ../RawImage\ 1.jpg 1   for upsample face detection

## Requirement
dlib 19.1.0<br />
opencv 2.4<br />
